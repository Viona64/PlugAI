Getting Started

Introduction to PlugAI

PlugAI is the unified AI API platform that lets developers and businesses integrate powerful AI capabilities into any application with a single API key and a few lines of code.

Version 2.1 is now available. See the Changelog for what's new.

Installation

Install the PlugAI SDK using your package manager:

pip install plugai
npm install @plugai/sdk
go get github.com/plugai/plugai-go

Authentication

All API requests require an API key. You can generate one from the Dashboard. Pass your key in the Authorization header:

Authorization: Bearer plug_sk_your_api_key_here
Never expose your API key in client-side code. Always use environment variables.

Chat API

Send a message to one of our language models and receive a response.

POST /v1/chat

Parameters

ParameterTypeRequiredDescription
modelstringYesThe model ID to use (e.g. plugai-gpt)
messagestringYesThe user message to send
temperaturefloatNoRandomness 0–2. Default: 0.7
streambooleanNoStream response tokens. Default: false
max_tokensintegerNoMax tokens in response. Default: 1024

Example Request

import plugai

client = plugai.Client(api_key="plug_sk_...")

response = client.chat.complete(
    model="plugai-gpt",
    message="Explain quantum computing simply.",
    temperature=0.7,
    max_tokens=512
)

print(response.text)
import PlugAI from '@plugai/sdk';

const client = new PlugAI({ apiKey: process.env.PLUGAI_API_KEY });

const response = await client.chat.complete({
  model: 'plugai-gpt',
  message: 'Explain quantum computing simply.',
  temperature: 0.7
});

console.log(response.text);
curl https://api.plugai.dev/v1/chat \
  -H "Authorization: Bearer plug_sk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "model": "plugai-gpt",
    "message": "Explain quantum computing simply.",
    "temperature": 0.7
  }'
